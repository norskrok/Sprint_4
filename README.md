# Sprint_4.4

## Описание проекта
Учебный проект в рамках спринта №4.  
Проект содержит автоматизированные тесты и собирается с помощью Maven.

## Используемые технологии

| Технология | Версия |
|------------|--------|
| Java       | 11     |
| Maven      | 3.11.0 |
| JUnit      | 4.13.2 |
| Selenium   | 4.39.0 |

## Структура проекта
- `src/main/java` — основной код проекта
- `src/test/java` — тесты
- `pom.xml` — конфигурация Maven

## Настройка проекта
Дополнительная настройка не требуется.  
Для работы проекта необходимо, чтобы были установлены:
- Java 11
- Maven

## Запуск тестов
Для запуска тестов выполните команду:

```bash
mvn clean test
